To get information on how to configure dhtmlxDiagram, explore our documentation: https://docs.dhtmlx.com/diagram.
You can import dhtmlxDiagram into your projects using yarn or npm package manager. To get the trial version of dhtmlxDiagram, run the following commands:

// yarn
yarn config set @dhx:registry https://npm.dhtmlx.com
yarn add @dhx/trial-diagram

Or

// npm
npm config set @dhx:registry https://npm.dhtmlx.com
npm i @dhx/trial-diagram
